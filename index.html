Urlizr

URL Shortener written in Python/Django

Download as .zip Download as .tar.gz View on GitHub

First of all copy the settings.py file to local_settings.py Don't forget to fill database acces in this file The you can start the dev server by launching: $ python manage.py runserver

This URL shortener also provide a useful API to integrate in your website You can retrieve shortened URL in raw, json or xml format just via this url: http:///api/translate/ You just have to pass POST parameters with url containing the url you want to shortened
